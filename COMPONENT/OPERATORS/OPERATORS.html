Arithmetic Operators

Comparison(Reletional) Operators

Relational Operators of Result always is true or false.

Reletional Operators

(<)< less than
(>) greater than
(==) equal
(<=)< less than or equal
(>=) greater than or equal
(!=) not equal
(===) equal value and same data type
(!==) not equal value and not same data type

logical Operators

logical Operators of Result always is true or false

logical Operators types

AND logical - (codition)&&(codition)
In AND logical Operators two both codition must be true like - (a<b)&&(a>b)
OR logical - (codition) || (codition)
In OR logical Operators any one codition must be true like - (a<b) || (a>b)
NOT logical - !(codition)
In NOT logical - if codition is false is result is true and codition is true is result is false like !(a<b)
